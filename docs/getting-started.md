## 2. Install the Plugin

After downloading:
1. Extract the ZIP file
2. Follow the installation instructions provided in the included PDF
3. Restart Rhino or Revit after installation

> Detailed installation guides for Rhino and Revit will also be available in separate documentation pages.

---

## 3. First Use

After installation, you can start using Termite directly from your CAD software.

1. Open **Rhino** or **Revit**
2. Launch the **Termite plugin** from the toolbar or panel
3. The plugin connects automatically to the **[Termite Cloud Dashboard](https://dashboard-ai.termite.tools/)**.


At this stage, no manual configuration is required.
